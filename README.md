# 北极冰基浮标数据可视化观测系统

这是一个面向北极冰基浮标遥测数据接收与展示的可视化观测系统。项目主要用于接收浮标通过数传模块回传的 GPS 位置及各类检测数据，并以地图轨迹、状态面板和数据记录的方式进行实时观测。

当前可运行的主程序位于 `mavlink_viewer/` 目录下，为基于 Electron + Vue 3 的桌面客户端。

## 项目简介

北极冰基浮标在野外部署后，会通过数传链路持续回传位置和传感器数据。本系统主要解决以下问题：

- 接收浮标设备通过串口数传链路回传的数据
- 对多发射端数据进行区分和实时展示
- 在地图上显示浮标位置与运动轨迹
- 展示 GPS、姿态、IMU、气压、温度等观测信息
- 将观测数据保存为 CSV 文件，便于后续分析与归档
- 支持北极/南极极地投影，适合高纬地区轨迹观测

## 主要功能

- 串口设备识别、连接与断开
- 双发射端数据接收与区分显示
- GPS 实时定位与轨迹绘制
- 常规地图投影与极地投影切换
- 在线地图与离线瓦片地图切换
- 姿态、IMU、气压、温度等状态面板展示
- 实时数据记录，导出为 CSV 文件

## 系统组成

当前项目主体位于 `mavlink_viewer/`：

- `src/main/`：Electron 主进程，负责串口通信、MAVLink 解析、离线瓦片服务、数据记录
- `src/renderer/`：Vue 前端界面，负责地图展示、数据面板、状态栏和交互控制
- `tiles/`：离线地图瓦片目录
- `test_serial_raw.js`：串口原始数据测试脚本
- `test_xbee_link.js`：数传链路测试脚本

## 当前支持的数据类型

系统当前已支持接收和展示以下类型的数据：

- GPS 经纬度、高度、卫星数、HDOP
- 姿态角：横滚、俯仰、偏航
- IMU：加速度、陀螺仪、磁力计
- 环境参数：气压、温度
- 心跳与基础遥测状态

## 快速开始

### 1. 环境要求

- Node.js 18 及以上
- npm 9 及以上
- Windows 环境下可用串口设备或数传模块

### 2. 安装依赖

```bash
cd mavlink_viewer
npm install
```

### 3. 启动开发环境

```bash
npm run dev
```

启动后会打开桌面客户端界面。

### 4. 构建 Windows 版本

```bash
npm run build:win
```

构建产物默认输出到：

```bash
mavlink_viewer/dist-electron/
```

## 使用流程

1. 启动程序并刷新串口列表
2. 选择连接的数传模块串口与波特率
3. 建立串口连接，开始接收浮标回传数据
4. 在地图区域观察实时位置与轨迹
5. 在右侧数据面板查看 GPS 与传感器状态
6. 需要保存数据时点击“开始记录”，系统会生成独立 CSV 文件

## 数据记录

程序支持将不同发射端的数据分别记录为 CSV 文件，便于后续整理、分析和科研使用。

输出文件名示例：

```text
mavlink_data_emitter1_2025-12-07T14-23-03.csv
mavlink_data_emitter2_2025-12-07T14-23-05.csv
```

## 地图显示说明

系统支持两类地图显示模式：

- 常规投影：适合全局或普通区域查看
- 极地投影：适合北极、南极等高纬区域轨迹观测

同时支持：

- 在线地图瓦片
- 本地离线瓦片地图

如果需要离线使用，可将瓦片按如下结构放置：

```text
tiles/{z}/{x}/{y}.png
```
